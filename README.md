# 🐔 Koko-chan: Tokyo Crossing (Rooster Fighter Edition)

Bem-vindo ao **Koko-chan: Tokyo Crossing**, um jogo arcade inspirado na estética do anime *Rooster Fighter (Keiji)*!
Ajude o galo lutador mais destemido de Tóquio a atravessar a movimentada avenida de Shibuya sem ser atropelado pelos carros e scooters ultra-rápidos!

---

## 🎮 Como Jogar Online (Web Version)

O jogo foi implantado na **Vercel** e pode ser jogado diretamente no seu navegador, no computador ou no celular:
🔗 **Jogue Online aqui:** [https://jogo-python.vercel.app](https://jogo-python.vercel.app) *(ou no link gerado na sua Vercel)*

### Controles (Web)
- **Teclado (PC)**: Use as setas do teclado (`▲`, `▼`, `◀`, `▶`) ou as teclas `W`, `A`, `S`, `D`.
- **Celular / Touch**: Use o gamepad virtual na tela (botões direcionais na parte inferior).

---

## 🐍 Como Jogar Localmente (Python Version)

Se preferir rodar o jogo offline em seu computador utilizando Python:

### Pré-requisitos
1. Ter o **Python 3** instalado em sua máquina.
2. Instalar a biblioteca **Pygame**:
   ```bash
   pip install pygame
   ```

### Executando o Jogo
Navegue até a pasta do projeto e execute:
```bash
python game.py
```

### Controles (Python)
- Use as setas do teclado ou as teclas `W`, `A`, `S`, `D` para desviar do tráfego.
- Aperte `SPACE` ou `ENTER` para iniciar o jogo ou reiniciar quando der Game Over.

---

## 🎨 Características do Jogo

- **Aparência de Anime**: Gráficos inspirados no galo Keiji de *Rooster Fighter*.
- **Efeitos de Impacto**: Tremor de tela (screen shake) e flashes vermelhos ao bater.
- **Sistema de Partículas**: Penas brancas voando ao pular ou colidir, faíscas de néon nas batidas e pétalas de Sakura (cerejeira) caindo suavemente.
- **Som Retro Retro-Sintetizado**: Sons gerados em tempo real (sem necessidade de arquivos MP3/WAV pesados) usando Web Audio API na Web e decodificadores internos no Pygame.
- **Recordes (High Scores)**: Armazenamento local do recorde para você sempre tentar se superar.
